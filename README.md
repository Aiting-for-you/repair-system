# 维修计价助手

这是一个为学校维修服务设计的计价系统Web应用，主要用于管理学校各类维修项目的价格计算和计价单生成。

## 功能列表

- 学校选择
- 维修项目管理
- 价格计算
- 计价单生成 
- 后台数据管理
  - 维修项目维护
  - 学校信息管理
  - 价格管理
  - 维修单批量导出
  - 用户账户管理 (可选)

## 技术栈

- 后端: Python (Flask)
- 前端: HTML, CSS, JavaScript
- 数据库: SQLite

## 如何运行

1.  安装依赖: `pip install -r requirements.txt`
2.  运行后端: `python app.py` (在 backend 目录)
3.  在浏览器中打开 `frontend/index.html`
